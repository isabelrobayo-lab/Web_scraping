Errores Priorizados (HUs)

HU-01 P0 Crítico

API de recomendados inaccesible (CORS + 404)

Zona: Home · Error: Cross-Origin Request Blocked: api-backend.ciencuadras.com/prod/recomendados/api/v1/recientes

Endpoint devuelve 404 y no incluye header Access-Control-Allow-Origin. La sección "inmuebles recientes" no carga; cascada de ERROR Error en main.js.

Abordaje: Verificar si el endpoint existe; si está deprecado eliminar la llamada; si existe configurar CORS. Añadir try/catch para que un fallo no rompa el render del Home.

HU-02 P1 Alto

Errores genéricos no manejados en resultados de búsqueda

Zona: Arriendo, Venta, Productos, Remates · Error: ERROR Error en result/4788.*.js

Excepción no capturada en el módulo de resultados (imágenes, filtros o datos vacíos). Posible degradación de UX.

Abordaje: Localizar chunk 4788; añadir error boundaries; implementar logging (Datadog/Sentry).

HU-03 P1 Alto

Cookies de terceros rechazadas por dominio inválido

Zona: Inmobiliarias · Error: Cookie "_fbp" y "_tt_enable_cookie" rejected for invalid domain

Scripts de Facebook Pixel y TikTok intentan setear cookies con dominio incorrecto. Tracking de conversiones afectado.

Abordaje: Revisar GTM/scripts; asegurar dominio coincida con ciencuadras.com.

HU-04 P1 Alto

CORS fallido en Taboola (publicidad)

Zona: Inmobiliarias, Constructoras, Productos · Error: Cross-Origin Request Blocked: cds.taboola.com

Widget de contenido patrocinado puede no cargar correctamente.

Abordaje: Contactar soporte Taboola; cargar script async/defer; envolver en try/catch.

HU-05 P2 Medio

Preload de imágenes ignorado

Zona: Arriendo, Venta, Productos · Falta as="image" o type en <link rel="preload">

Abordaje: Añadir as="image" y type="image/jpeg"; validar con Lighthouse.

HU-06 P2 Medio

Fuente con glyph bbox incorrecto

Zona: Home · font-family: "ciencuadras"

Abordaje: Regenerar fuente o reemplazar por SVG.

HU-07 P2 Medio

PerformanceObserver con entryTypes no soportados

Zona: Home · layout-shift, longtask en Firefox

Abordaje: Verificar PerformanceObserver.supportedEntryTypes antes de observar.

HU-08 P2 Medio

Error no capturado en Host Listener (menú privado)

Zona: Home · UNCAUGTHER ERROR HOST LISTENER PRIVATE MENU

Abordaje: Buscar en código; añadir validaciones null/undefined; try/catch con logging.

HU-09 P3 Bajo

Timeouts en carga de páginas

Zona: Inmobiliarias, Blog, Auth · networkidle timeout 30s

Abordaje: Revisar polling/websockets; optimizar APIs; lazy loading de scripts terceros.
