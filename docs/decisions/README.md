# Decisiones Técnicas (ADRs)

Este directorio almacena el **por qué** de las decisiones técnicas del proyecto.

## Formato

Cada decisión se documenta en un archivo con el patrón:

- `YYYY-MM-DD-nombre-corto.md`
- O `nombre-descriptivo.md` si no hay fecha específica
